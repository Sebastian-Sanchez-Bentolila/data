Conclusiones

En este trabajo, se desarroll un modelo de regresin lineal mltiple para predecir el ndice de desempeo (Performance Index) de estudiantes basado en varias variables de entrada, como horas de estudio, calificaciones previas, actividades extracurriculares, horas de sueo y nmero de exmenes practicados. Se realizaron las siguientes etapas:

- Lectura y Preprocesamiento de Datos: Se cargaron los datos y se exploraron inicialmente para comprender su estructura.

- Normalizacin de Variables: Se normalizaron las variables de entrada para asegurar que todas tuvieran la misma escala.

- Transformacin de Variables Categricas: Se transform la variable categrica "Actividades Extracurriculares" en una variable numrica.

- Separacin de Datos: Se dividieron los datos en conjuntos de entrenamiento y prueba utilizando train_test_split.

- Construccin y Entrenamiento del Modelo: Se desarroll un modelo de regresin lineal mltiple con los datos de entrenamiento.

- Evaluacin del Modelo: Se evalu el desempeo del modelo utilizando mtricas como el Error Absoluto Medio, Error Cuadrtico Medio, Puntuacin R2 y Puntuacin de Varianza Explicada. Adems, se realizaron grficos de dispersin de datos pronosticados vs. datos de prueba y de los errores del modelo.

Los resultados muestran que el modelo es capaz de capturar la relacin entre las variables explicativas y el ndice de desempeo con un nivel razonable de precisin. Los grficos de dispersin y de errores sugieren que aunque el modelo es eficaz, existen algunas desviaciones.
