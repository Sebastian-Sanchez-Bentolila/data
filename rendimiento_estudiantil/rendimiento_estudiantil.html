Conclusiones¶

En este trabajo, se desarrolló un modelo de regresión lineal múltiple para predecir el índice de desempeño (Performance Index) de estudiantes basado en varias variables de entrada, como horas de estudio, calificaciones previas, actividades extracurriculares, horas de sueño y número de exámenes practicados. Se realizaron las siguientes etapas:

- Lectura y Preprocesamiento de Datos: Se cargaron los datos y se exploraron inicialmente para comprender su estructura.

- Normalización de Variables: Se normalizaron las variables de entrada para asegurar que todas tuvieran la misma escala.

- Transformación de Variables Categóricas: Se transformó la variable categórica "Actividades Extracurriculares" en una variable numérica.

- Separación de Datos: Se dividieron los datos en conjuntos de entrenamiento y prueba utilizando train_test_split.

- Construcción y Entrenamiento del Modelo: Se desarrolló un modelo de regresión lineal múltiple con los datos de entrenamiento.

- Evaluación del Modelo: Se evaluó el desempeño del modelo utilizando métricas como el Error Absoluto Medio, Error Cuadrático Medio, Puntuación R2 y Puntuación de Varianza Explicada. Además, se realizaron gráficos de dispersión de datos pronosticados vs. datos de prueba y de los errores del modelo.

Los resultados muestran que el modelo es capaz de capturar la relación entre las variables explicativas y el índice de desempeño con un nivel razonable de precisión. Los gráficos de dispersión y de errores sugieren que aunque el modelo es eficaz, existen algunas desviaciones.
